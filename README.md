# OMNI Technical Challenge 💸

API RESTful para simular um sistema simples de **transações monetárias entre usuários**, desenvolvida com [NestJS](https://nestjs.com/).

---

## 📌 Funcionalidades

- **Cadastro de usuários**
- **Login de usuários com geração de token JWT**
- **Transferência de saldo entre usuários**
- **Consulta de todos os usuários (com seus saldos)**

---

## 🔧 Tecnologias Utilizadas
- Node versão 22.15
- [NestJS](https://nestjs.com/)
- [TypeORM](https://typeorm.io/)
- SQLite
- Docker
- Testes com Jest

---

## Instale as dependências

```npm install```

## Inicie o servidor

```npm run start:dev```
